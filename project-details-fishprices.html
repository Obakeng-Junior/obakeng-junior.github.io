FishPrices - Fish Catch Price Calculator

Web Application Completed: 2023
ASP.NET Core MVC C# Bootstrap SQL Server Entity Framework JavaScript

ASP.NET Core MVC web application designed specifically for fishermen to calculate the total value of their daily catch. The application allows users to input kilograms caught for each fish type, automatically calculates prices based on current market rates, and provides detailed itemized summaries with grand totals.

This application solves a real-world problem for local fishing communities by eliminating manual calculations and reducing errors in pricing. The system is designed with simplicity and efficiency in mind, ensuring even users with minimal technical experience can quickly calculate their earnings.

Key Features:

  • Interactive fish selection with current market prices
  • Dynamic calculation form with real-time price updates
  • Itemized summary with detailed breakdown per fish type
  • Grand total calculation with tax considerations
  • Print and export functionality for calculations
  • Admin panel for managing fish types and price updates
  • Responsive design optimized for mobile devices
  • Calculation history and saved sessions
  • Multi-language support (English & local dialects)
  • Offline calculation capability

Business Impact:

The application has been adopted by several local fishing cooperatives, reducing calculation errors by 95% and saving an average of 2 hours per day for each fisherman. The system also provides valuable data analytics for tracking fishing trends and market price fluctuations.
